# ATM_SIMULATION

A simple and modular Python-based ATM Simulation that allows users to create accounts, login securely, deposit money, withdraw money, check balance, view transaction history, and change PIN. Data is stored in JSON for persistence and reliability.

Features
- Create New Account
- Secure Login with 4-digit PIN
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Mini Statement (Last 10 Transactions)
- Change PIN
- JSON-based Data Storage
- Fully Menu-Driven CLI Interface

Technologies Used
- Python 3
- JSON Storage
- Datetime Module
- OS Module
  
Testing Instructions
- Create a new user account
- Login using Account Number & PIN
- Perform deposit, withdrawal
- Check balance
- View mini statement
- Change PIN and verify
- Check JSON file updates
