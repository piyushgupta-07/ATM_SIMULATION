# ATM Simulation Project Statement

## Project Title
**ATM Simulation System**

## Developed By
**Piyush Gupta**

## Overview
This project is a simple command-line based ATM Simulation System written in Python. It allows users to create accounts, log in, deposit money, withdraw funds, check balance, view mini statements, and change their PIN.

## Features
- Create New Account with unique account number
- Secure 4-digit PIN authentication
- Deposit and Withdrawal functionality
- Balance enquiry
- Mini statement showing last 10 transactions
- Change PIN option
- Data stored in a JSON file (`atm_db.json`)

## Files Included
- **main.py** – Contains full ATM simulation code
- **atm_db.json** – Database file for storing accounts and transactions
- **statement.md** – Project description file (this file)

## How It Works
1. Run the script.
2. Create a new account or log in with an existing one.
3. Perform ATM operations using the user menu.
4. All data is saved automatically.

## Requirements
- Python 3.x
- No external libraries required (uses only built-in modules)

## Usage
To run the project:
```bash
python main.py
```
